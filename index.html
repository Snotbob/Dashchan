Dashchan
Android imageboard client

Welcome to the Dashchan GitHub page.

Dashchan is Android client for multiple imageboards.

Downloads

You can view list of packages and download them here.

You must install the client package and extensions. Every extension supports one imageboard.

Imageboard Implementation

Javadoc for Dashchan Library is available here.

There is 4 basic classes that you must extend and make implementation of some methods:

  • ChanConfiguration: determines capabilities and stores configuration data.
  • ChanPerformer: performs connectivity with imageboard.
  • ChanLocator: parses and builds links.
  • ChanMarkup: manages post markup.

Also you must extend the following models:

  • ThreadModel: holds thread data.
  • PostModel: holds post data.

If your imageboard don't support API, you must parse HTML for boards, threads and posts pages. You can do it with GroupParser or with imported library. Don't forget to move library to another package with unique name! Different versions of libraries can cause conflicts if somebody imported this library too.

You can find implementation examples on my GitHub page. There are some finished extensions that have the same imageboard engine like on yours one. It's may be enough to fork extension with some changes.
