Class Overview

Provides performing connectivity with chan.

By default you must implement the following methods:

  • readThreads(ReadThreadsData)
  • readPosts(ReadPostsData)
  • readBoards(ReadBoardsData)

If you request some features, you also must implement these methods:

FeatureMethod
FEATURE_SINGLE_BOARD_MODE readSinglePost(ReadSinglePostData)
FEATURE_SEARCH readSearchPosts(ReadSearchPostsData)
FEATURE_USER_BOARDS readUserBoards(ReadBoardsData)
FEATURE_POPULAR_THREADS readGlobalThreads(ReadGlobalThreadsData)
FEATURE_WATCHER readPostsCount(ReadPostsCountData)
FEATURE_CAPTCHA_PASS checkCaptchaPass(CheckAuthorizationData)
FEATURE_USER_AUTHORIZATION checkUserAuthorization(CheckAuthorizationData)
FEATURE_SEND_POST readCaptcha(ReadCaptchaData)
FEATURE_SEND_POST sendPost(SendPostData)
FEATURE_DELETE_POSTS sendDeletePosts(DeletePostsData)
FEATURE_SEND_REPORT sendReport(SendReportData)

public ChanPerformer.CaptchaResult readCaptcha (ChanPerformer.ReadCaptchaData data)

Calls when client request chan to read captcha.

This method must be overridden if chan supports feature FEATURE_SEND_POST.

This method must return ChanPerformer.CaptchaResult with captcha data.

If your chan uses custom captcha, the result must hold image.

In the case of Yandex Captcha, resulting captchaData must contain captcha key by CAPTCHA_KEY.

In the case of Google reCAPTCHA, resulting captchaData must contain reCAPTCHA API key by RECAPTCHA_API_KEY.

If your captcha has short lifetime, you can check mayShowLoadButton. If this argument equals true, the result may hold NEED_LOAD. This will show "Click to load captcha" button for user.

public void sendDeletePosts (ChanPerformer.DeletePostsData data)

Calls when client request chan to delete posts.

This method must be overridden if some boards support feature FEATURE_DELETE_POSTS.

postNumbers contains several post numbers. You must perform deleting all of these posts. You must throw ApiException only in case if nothing was deleted. In other cases you must return and throw nothing.

You can throw ApiException with DELETE_* error types.
