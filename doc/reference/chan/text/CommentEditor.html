public final void addSimilarDependency (int what, int similar)

Add a new similar dependency. That means, that similar is similar to what, but longer.

For example, bold ** and italic * in Wakaba Mark: bold is similar to italic, but longer. So you must pass TAG_ITALIC as first argument and TAG_BOLD as second one.

Parameters
what Tag.
similar Similar to what tag, but longer.

public void addTag (int what, String open, String close)

public CommentEditor.FormatResult formatSelectedText (Editable editable, int what, int start, int end)

This method calls every time user click to markup button. You must handle it and transform editable, including selection position, and then return new selection position with CommentEditor.FormatResult instance.
